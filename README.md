# Zed Theme Utils [WIP]

A work-in-progress tool to manage Zed editor themes.

- [x] Zed theme file parser
- [x] Schema validation for different versions
- [ ] Theme migration between schema versions
- [x] Diff visualization between theme versions
- [x] Italicization with customizable keys
- [ ] VSCode to Zed transformer with smart color mapping
- [x] API and CLI interfaces
- [x] TypeScript-first with full type definitions
- [ ] Extensible architecture for custom migrations and transformations
